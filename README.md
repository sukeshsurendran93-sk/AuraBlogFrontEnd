# Aura.Log

A premium blog platform built with React, Vite, Redux Toolkit, and Tailwind CSS. Features a rich text editor, AI-powered writing assistant, nested comments, bookmarking, dark mode, and responsive design.

## Features

- Rich Text Editor — Full WYSIWYG editing with react-quill-new, supporting headings, lists, blockquotes, code blocks, links, and images
- AI Co-Pilot — Auto-summarize articles, get content suggestions, and check grammar/readability scores using NLP libraries (compromise, write-good, text-readability)
- Nested Comments — Threaded discussions with reply support and like counts
- Bookmarking — Save articles for later; persisted to localStorage
- Search & Filter — Real-time search across titles, content, and tags; filter by category or bookmarked posts
- Dark Mode — System-aware theme toggle with smooth transitions
- Responsive Design — Mobile-first layout that works on all screen sizes
- Accessibility — ARIA labels, keyboard navigation, and screen reader support
